# Alphabet Voting App
Quick IOS app that uses ListViews and unique button functionality

# Get It Up
- Clone this repo
- 'npm install' from terminal
- Build as an IOS project
- Let me know how it goes


# Extras
- Updates to Database are permanet
- Leaderboard accessible through web App as well as in Native App
- List only loads 10 characters at a time(an loads more on scroll)
- All features and functions run quickly (Capping at O(n))
